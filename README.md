
#  Cybersecurity Portfolio — Aylan HANIFI

## Projet — Audit GRC Cloud Azure

**Type :** Projet personnel — Évaluation des risques cloud  
**Outils :** Microsoft Azure, NIST CSF 2.0, CIS Controls v8, ISO 27001, ANSSI

### Contexte
Simulation d'un audit GRC sur un compte de stockage Azure 
volontairement mal configuré, reproduisant les erreurs les plus 
courantes rencontrées en entreprise.

### Ce qui a été réalisé
- Identification de 5 actifs critiques et 5 vulnérabilités
- Quantification des risques par méthode Probabilité × Impact
- 3 risques CRITIQUES (score 20/25) identifiés
- Mapping de 8 contrôles aux frameworks NIST CSF, CIS Controls, 
  ISO 27001 et recommandations ANSSI
- Rédaction d'un rapport GRC complet avec plan de remédiation priorisé
